# Maze3d

# Prerequisites #

* Maven (3.2.1+) (http://maven.apache.org/download.cgi)
* JDK 1.8+ (http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)

(Tested with Oracle JDK)

# Building #

'cd' to the root of the project

``` bash
$ mvn package
```

# Running #

``` bash
$ mvn exec:java
```
